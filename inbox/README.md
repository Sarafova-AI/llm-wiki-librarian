# Inbox

This folder is the staging area for sources that should enter the wiki.

Use one file per source.

## Minimal Metadata

Put this at the top if you know it:

```text
source_provider:
source_surface:
source_date:
native_url:
project:
title:
privacy_level:
```

Then paste the raw chat, note, article text, or terminal transcript below.

## Filename Shortcut

If you do not want to type metadata, use:

```text
YYYY-MM-DD-provider-surface-project-short-topic.md
```

Examples:

```text
2026-05-20-claude-desktop-research-planning.md
2026-05-20-codex-terminal-build-notes.md
2026-05-20-chatgpt-mobile-article-ideas.md
```

Then tell your local AI agent:

```text
Librarian, process the inbox.
```
